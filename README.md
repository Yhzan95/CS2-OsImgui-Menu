
# <p align="center">CS2 Imgui Menu</p>
  
This project is a simple overlay application using OsImGui to display an interactive menu in Counter-Strike 2.
It allows you to show a configurable menu directly inside the game and automatically closes when the CS2 window is closed.
      <p align="center">
  <img src="https://github.com/Yhzan95/CS2-Imgui-Menu/blob/main/Menu.png" width="500" style="border: 1px solid #000;"/>
</p>
## 🧐 Features    
- Fully configurable Imgui menu
- Close automatically when cs2 shutdown
- On/Off Toggle menu
        
## 🛠️ Install Dependencies
```bash
Net Framework v4.7.2
```
🧑🏻‍💻 Usage
```js
1) Open Cs2.Exe
2) Open your exe file
3) Menu will pop automatically, Press Insert for toggle On/Off
4) Profit
```
